Languages

Create your own language

Creating a new language is very simple. This can be achieved adding a new entry to the W3FCT.lang object. 

//add a new custom language
W3FCT.lang.mySpanishLang = {
  days: ['DOM', 'LUN', 'MAR', 'MIE', 'JUE', 'VIE', 'S√ÅB'],
  months: ['ENE', 'FEB', 'MAR', 'ABR', 'MAY', 'JUN', 'JUL', 'AGO', 'SET', 'OCT', 'NOV', 'DIC'],
  ampm: ['a.m.', 'p.m.']
};

//set your previously created custom language
mytarget.W3FClockTimer({
    lang: 'mySpanishLang',
});
